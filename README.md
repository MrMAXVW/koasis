# KOASIS Company Limited Website 🚀

This is the official single-page website for **KOASIS Company Limited**, built with a focus on speed, AI-driven solutions, and a "Customer First" philosophy. It’s cleaner than your workspace and faster than your morning coffee.

---

## ✨ Features
*   **Modern AI Stack**: Built with React and Tailwind CSS for that smooth, premium feel.
*   **Bilingual Support**: Instant toggle between Traditional Chinese (Cantonese localized) and English—no text labels, just clean icons.
*   **Interactive UI**: Features a dynamic floating background orb that follows your cursor—because static is boring.
*   **Fully Responsive**: Looks great on everything from a massive 4K monitor to the phone you're hiding under your desk.
*   **Smart Contact**: Direct `mailto:` integration because no one has time to copy-paste email addresses in 2026.

## 🛠️ Tech Used
*   **Frontend**: React (18.x) via CDN.
*   **Styling**: Tailwind CSS for rapid, utility-first UI development.
*   **Icons**: Google Material Symbols.
*   **Assets**: SVG-based branding for crisp visuals at any scale.

## 🚀 Quick Start
1.  Clone the repo to your local machine.
2.  Open `index.html` in any modern web browser.
3.  Deploy to **GitHub Pages** for instant hosting.
